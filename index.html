Neboj se, programování není žádná černá magie – je to spíš jako skládání kostek nebo malování obrázků, jen pomocí kódu. Tento web ti ukáže základy tří hlavních jazyků, které tvoří moderní web: HTML, CSS a JavaScript.

HTML jako kostra webu

HTML je značkovací jazyk, který určuje strukturu webové stránky. Každý prvek stránky je zapsán pomocí značky (tagu), kterou si můžeme představit jako obal – například <div> je jako malá krabička, do které můžeme vložit obsah. Stejně fungují i ostatní značky – každá má svůj účel a společně tvoří kostru stránky.

Značky

Jak už jsme si řekli, HTML stránky se skládají ze značek, což jsou vlastně takové obaly, do kterých vkládáme obsah. Například <p> slouží k označení odstavce textu, takže text uvnitř tohoto obalu bude považován za paragraf. Další běžné značky jsou <h1> až <h6> pro nadpisy, <img> pro obrázky, nebo <div>, což je univerzální obal pro bloky obsahu. Každá značka má svůj účel, ale všechny fungují podobně – obalují a strukturalizují obsah stránky.

Seznamy a obrázky

Pro tvorbu seznamů používáme značky <ul> (nečíslovaný seznam) a <ol> (číslovaný seznam), přičemž jednotlivé položky seznamu jsou uvnitř <li> tagu. Například: 

    <ul>
      <li>První položka</li>
      <li>Druhá položka</li>
      <li>Třetí položka</li>
    </ul>

Pro obrázky používáme samouzavírací značku <img>. Hlavní atributy jsou src (cesta k obrázku) a alt (popis obrázku). Příklad: 

    <img src="obrazek.jpg" alt="Popis obrázku">

Tabulky

Pro tabulky se používají značky <table> (celá tabulka), <tr> (řádek), <th> (hlavička sloupce) a <td> (buňka dat). Například: 

    <table>
      <tr>
        <th>Jméno</th>
        <th>Věk</th>
      </tr>
      <tr>
        <td>Alice</td>
        <td>25</td>
      </tr>
      <tr>
        <td>Bob</td>
        <td>30</td>
      </tr>
    </table>

Struktura stránky

Každá HTML stránka má základní strukturu rozdělenou na <head> a <body>. V <head> se nachází informace o stránce, jako je název, meta tagy nebo odkazy na styly.Celkově se tady nachází část kterou uživatel nevidí. <body> naopak obsahuje vše, co se zobrazuje uživateli – texty, obrázky, seznamy, tabulky a další obsah. Správné oddělení těchto částí pomáhá udržet stránku přehlednou a funkční.

CSS – styl a design webu

Abychom si lépe představili, jak funguje design webu, můžeme si to přirovnat ke stavbě domu. HTML prvky jako <h1>, <p>, <div> jsou jako jednotlivé části stavby – stěny, místnosti nebo okna. CSS třídy jako .mujstyl jsou naopak „výzdoba“ a vzhled: určují barvy, velikosti, zarovnání a celkový styl. HTML tedy tvoří strukturu a CSS určuje, jak bude stránka vypadat.

Komponenty (HTML prvky)

HTML prvky jako <h1>, <p> nebo <ul> se používají k označení významu obsahu. Říká se tomu sémantika. Například <h1> je hlavní nadpis stránky, <p> představuje odstavec textu a <h2> až <h6> označují podnadpisy různých úrovní. Tyto značky mají svou přirozenou stylovou váhu – například nadpisy jsou větší než odstavce. Pokud má stránka současně přiřazený CSS styl, platí, že CSS má přednost před výchozím (defaultním) vzhledem prvků, ale sémantický význam HTML zůstává.

Třídy (CSS classes)

Třídy jsou způsob, jak dát prvkům vlastní vzhled. V HTML se přidávají pomocí atributu class="" a v CSS se zapisují tečkou, například .mujstyl. Díky třídám můžeme měnit barvy, velikosti, rozložení i mezery kolem prvků. Mezi základní CSS vlastnosti patří:

  • color – barva textu
  • background-color – barva pozadí
  • font-size – velikost textu
  • margin – vnější odsazení (mezera kolem prvku)
  • padding – vnitřní odsazení (mezera uvnitř prvku)
  • border – okraj prvku
  • text-align – zarovnání textu

Třídy jsou velmi flexibilní – stejný styl můžeš použít na více prvcích najednou, což pomáhá udržet web konzistentní, přehledný a snadno upravitelný. Díky tomu máme nad vzhledem webu úplnou kontrolu.

JavaScript – život a interaktivita webu

Zatímco HTML tvoří strukturu stránky a CSS její vzhled, JavaScript přidává webu život a umožňuje mu reagovat na uživatele. JavaScript dokáže měnit obsah stránky, ovládat prvky, reagovat na kliknutí, zobrazovat animace nebo pracovat s daty. Můžeme si ho představit jako mozek celé stránky – rozhoduje, co se má stát a kdy.

JavaScript pracuje s takzvaným DOMem, což je vnitřní reprezentace HTML, kterou si prohlížeč vytváří. Díky tomu může měnit text, obrázky, barvy, skrývat prvky nebo otevírat menu. I jednoduché skripty dokážou udělat web výrazně interaktivnější, a většina moderních webů jeho funkce využívá každý den.
